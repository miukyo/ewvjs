# my-ewvjs-app

A ewvjs application.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the application:
```bash
npm start
```

3. Package the application:
```bash
npm run package
```

## Project Structure

- `app.js` - Main application entry point
- `assets/` - Static assets (images, fonts, etc.)
- `package.json` - Project configuration
